The Framework

How it Works

01 Content Identification

Assigns a persistent Identification unique identifier to each creative asset, enabling tracking and analysis across channels.

02 Exposure

Tracks content exposure using pixel, server-to-server, or clean room to capture device Identifications.

03 Identity Match

Matches the exposure data to individual user identities deterministically via verified identity graphs.

04 Attribution

Proprietary Machine Learning assigns outcomes to the specific content that drove the actions.

05 Outcome

Measures outcomes such as site visits, brand lift, and conversions associated with the creative content.

06 Optimization

Machine Learning scores content performance to provide actionable media buy recommendations.
